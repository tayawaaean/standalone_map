Interactive Map - Embedding Examples

Example 1: Embedded Map (iframe)

This is how the map looks when embedded via iframe:

Code to use:

<iframe src="interactive-map.html" 
        width="100%" 
        height="600" 
        frameborder="0" 
        style="border-radius: 16px;">
</iframe>

Example 2: Direct Code Injection

For platforms that support direct HTML injection, simply copy the entire contents of interactive-map.html and paste it into your page.

Steps:

  1. Open interactive-map.html in a text editor
  2. Copy all the code (Ctrl+A, Ctrl+C)
  3. Paste into your website's HTML editor or custom code section

Platform-Specific Instructions

Wix

  1. Add an "HTML Embed" element to your page
  2. Click "Enter Code"
  3. Paste the entire HTML code from interactive-map.html
  4. Adjust the element size in the Wix editor

WordPress

  1. Add a "Custom HTML" block to your page
  2. Paste the HTML code
  3. Or use a plugin like "Insert Headers and Footers" for site-wide embedding

Squarespace

  1. Add a "Code Block" to your page
  2. Paste the HTML code
  3. The map will render automatically
